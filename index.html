What is the XSEDE User Portal?

  • Point of entry for new users to XSEDE
  • Increase productivity by offering services, support and documentation.
  • Focus on USER services
  • Accomplish these goals through multiple user interfaces:
    • Full services web site providing a rich user experience - http://portal.xsede.org
    • Mobile web interface for XUP on the go: http://mobile.xsede.org
    • Native iOS and Android applications (coming soon!)
